# Contributing to Talawa API

Thank you for your interest in contributing to Talawa API. Regardless of the size of the contribution you make, all contributions are welcome and appreciated.

## Guidelines

Please refer to the following document for **Talawa contribution guidelines**:

- [Talawa Contributing Guide](docs/docs/docs/developer-resources/getting-started/contributing.md)

You may also read the general Palisadoes contributing guidelines:

- [Palisadoes Contributing Guide](https://developer.palisadoes.org/docs/contributor-guide/contributing)

## Making Contributions

For a complete guide on how to use Git while contributing, please refer to the Palisadoes Git Guide:

- [Palisadoes Git Guide](https://developer.palisadoes.org/docs/git-guide/introduction/quickstart/)
