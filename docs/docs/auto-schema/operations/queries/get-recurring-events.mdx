  startOpen = false,

Fetches all recurring event instances that belong to a specific base recurring event template.

```graphql
getRecurringEvents(
  baseRecurringEventId: ID!
  includeCancelled: Boolean
  limit: Int
  offset: Int
): [Event!]
```
