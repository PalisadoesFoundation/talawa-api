Input type for querying metrics data. Supports time range filtering, operation type filtering, and duration-based filtering.

```graphql
input MetricsInput {
  endTime: DateTime!
  includeCacheMetrics: Boolean!
  maxDuration: Float
  minDuration: Float
  operationType: String
  startTime: DateTime!
}
```
