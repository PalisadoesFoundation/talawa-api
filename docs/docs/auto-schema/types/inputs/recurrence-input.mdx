Input type for defining recurrence rules for events. Must specify exactly one end condition: endDate, count, or never.


```graphql
input RecurrenceInput {
  byDay: [String!]
  byMonth: [Int!]
  byMonthDay: [Int!]
  count: Int
  endDate: DateTime
  frequency: Frequency!
  interval: Int
  never: Boolean
}
```
